시작 AI 블로그

AI 도구들을 활용해 일상 속 문제를 해결하고, 생산성을 높이고, 개인 프로젝트를 만들어가는 과정을 기록하는 블로그를 시작합니다.

왜 이 블로그인가

Claude Code, ChatGPT, Cursor 같은 AI 도구들을 써보면서 느낀 것들—잘 되는 것, 안 되는 것, 새로 발견한 방법들—을 그냥 흘려보내기 아깝다는 생각이 들었습니다. 잘 정리해두면 나중에 나 자신에게도, 비슷한 관심사를 가진 분들에게도 도움이 될 것 같아서 시작합니다.

다룰 내용

  • AI 개발 실험 — 코딩 자동화, 프롬프트 엔지니어링, 소형 프로젝트 제작기
  • 생산성 도구 — 워크플로 개선, 자동화 스크립트, 도구 리뷰
  • 취미 프로젝트 — 개인적으로 만들거나 배우는 것들

블로그 자체도 AI로

이 블로그 자체도 Claude Code와 함께 만들었습니다. GitHub Pages에서 빌드 도구 없이 HTML/CSS/JS만으로 운영할 수 있도록 설계했습니다. 포스트를 추가하거나 디자인을 바꿀 때도 Claude Code에게 요청하면 됩니다.

앞으로 자주 업데이트하겠습니다. 👋
